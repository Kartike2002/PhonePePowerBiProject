# 📊 PhonePe Transaction Dashboard (Power BI)

## 📌 Project Overview
This project is a **Power BI Dashboard** built using PhonePe transaction data to analyze digital payment trends across India.  
It provides meaningful insights into transaction volume, value, and user behavior through interactive visualizations.

---

## 🚀 Features
- 📍 State-wise transaction analysis  
- 💰 Total transaction value & count  
- 📈 Year-wise growth trends  
- 🧑‍🤝‍🧑 User insights and engagement metrics  
- 📊 Interactive filters and slicers  
- 📌 Clean and user-friendly interface  

---

## 🛠️ Tools & Technologies Used
- Power BI – Data visualization  
- Excel / CSV – Data source  
- GitHub – Version control

## 📊 Key Insights
- High transaction volume observed in metropolitan states  
- Rapid growth in digital payments over time  
- Increasing adoption in tier-2 and tier-3 cities  
- Noticeable seasonal spikes in transactions

## ⚙️ How to Use
1. Download the `.pbix` file from this repository  
2. Open it in Power BI Desktop  
3. Load or refresh the dataset  
4. Explore the dashboard using filters and visuals
